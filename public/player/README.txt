Nuevo plugin for Videojs framework version 7.x

Video.js player file used in `examples` directory is "video.min.js", located in "videojs" directory.
This is video.js minimized script, version 8.4.3 released 2023-06-06.
Full versions of video.js files are located in the same directory.


You may also use any CDN hosted video.js version above 8.0. Please note that delivered by Nuevodevel.com video.js scripts are slightly modified for better experience when the user drag and move playing progress bar.


Upload "videojs8" directory to your server.
Navigate to "videojs7/examples/index.html" for multiple HTML examples of videojs v7 and Nuevo v11 covering most popular videojs and nuevo plugin usage.
